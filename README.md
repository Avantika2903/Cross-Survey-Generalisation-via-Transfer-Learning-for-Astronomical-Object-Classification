# Cross-Survey-Generalisation-via-Transfer-Learning-for-Astronomical-Object-Classification

Overview

This research project investigates the effectiveness of cross-survey transfer learning for astronomical object classification using deep learning.

Modern astronomical surveys such as SDSS and DESI observe the same universe but differ in instrumentation, image characteristics, observational conditions, and data distributions. These differences create a domain shift that can significantly affect the performance of machine learning models when applied across surveys.

The goal of this project is to evaluate how well knowledge learned from one survey can be transferred to another using transfer learning techniques.

Specifically, the study focuses on the classification of:

Galaxies
Stars
Quasars (QSOs)

using astronomical image data and convolutional neural networks.
